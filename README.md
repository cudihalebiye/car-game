# Araba Yarış Oyunu 🚗

Python ve Pygame kullanılarak geliştirilmiş basit bir araba yarış oyunudur.

## Özellikler

* Hareket eden arabalar
* Puan sistemi
* Artan hız
* Yağmur efekti
* Oyun bitiş ekranı
* Ses efektleri

## Kullanılan Teknolojiler

* Python
* Pygame

## Geliştirici

cudihalebiye
## Oyunu Çalıştırma

Projeyi çalıştırmak için:

```bash
pip install pygame
python main.py
```

Eğer dosya src/cekirdek klasöründeyse:

```bash
cd src/cekirdek
python main.py
```
